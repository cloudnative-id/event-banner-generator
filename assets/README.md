# Assets Directory

This directory contains images used in the banner generation.

## Directory Structure

- `logos/` - Logo images (Kubernetes Indonesia, sponsors, etc.)
- `photos/` - Speaker/presenter photos

## Adding Images

1. **Logos**: Place logo images in the `logos/` directory
   - Recommended format: PNG with transparency
   - Recommended size: Square images work best

2. **Photos**: Place speaker photos in the `photos/` directory
   - Recommended format: JPG or PNG
   - Recommended size: At least 300x300 pixels
   - Square or portrait orientation works best

## Referenced Images

Make sure to add the following images referenced in the example YAML files:

### Logos
- `logos/cncf-indonesia-logo.png` - Main Kubernetes Indonesia logo
- `logos/sponsor-logo.png` - Sponsor logo (if applicable)

### Photos (for examples)
- `photos/john-doe.jpg`
- `photos/prasta-maha.jpg`

You can use placeholder images or your actual event photos.
